# 🕵️ ESP32 Spy Audio Recorder

[![ESP32](https://img.shields.io/badge/ESP32-Project-blue)](https://www.espressif.com/)
[![Telegram Bot](https://img.shields.io/badge/Telegram-Bot-26A5E4)](https://core.telegram.org/bots)

[![Arduino](https://img.shields.io/badge/Arduino-IDE-00979D)](https://www.arduino.cc/)

## 📋 Описание проекта

Устройство скрытого аудиомониторинга на базе ESP32, которое автоматически записывает звук при обнаружении речи и отправляет аудиофайлы в Telegram бот.

```mermaid
graph LR
    A[🎤 Микрофон] --> B[ESP32]
    B --> C[Wi-Fi]
    C --> D[🤖 Telegram Bot]
    D --> E[📱 Твой Telegram]
```

## ✨ Возможности

- 🎯 **Автоматическая запись** при обнаружении звука выше порога
- ⏱️ **Запись 3 секунды** при каждом срабатывании
- 📤 **Автоматическая отправка** в Telegram бот
- 🎚️ **Автоматическая калибровка** уровня шума
- 💡 **Светодиодная индикация** во время записи
- 🔒 **Защищенное соединение** с Telegram API

## 🔧 Аппаратное обеспечение

### Необходимые компоненты

| Компонент | Количество | Примечание |
|-----------|------------|------------|
| ESP32 Dev Board | 1 | Любая плата с WiFi |
| MAX9814 | 1 | Модуль микрофона с усилителем |
| LED | 1 | Встроенный или внешний |
| Провода | неск. | Для подключения |
| Макетная плата | 1 | Опционально |

### Схема подключения

```
MAX9814 → ESP32
----------------
VDD     → 3.3V
GND     → GND
OUT     → GPIO34

LED (опционально)
----------------
+       → GPIO2
-       → GND
```

## 📦 Установка и настройка

### 1. Создание Telegram бота

1. Найди в Telegram [@BotFather](https://t.me/botfather)
2. Отправь команду `/newbot`
3. Следуй инструкциям для создания бота
4. Сохрани полученный **токен**

### 2. Получение Chat ID

1. Найди [@userinfobot](https://t.me/userinfobot)
2. Отправь `/start`
3. Скопируй свой **ID**

### 3. Настройка кода

Замени значения в начале файла:

```cpp
// ===== НАСТРОЙКИ =====
#define WIFI_SSID     "твоя_wifi_сеть"     // Имя WiFi сети
#define WIFI_PASS     "твой_пароль"        // Пароль WiFi
#define BOT_TOKEN     "твой_токен_бота"    // Токен от BotFather
#define CHAT_ID       "твой_telegram_id"   // ID от userinfobot
```

### 4. Установка библиотек

**Arduino IDE:**
1. Инструменты → Управлять библиотеками
2. Найти и установить: `Universal Telegram Bot` (автор: Brian Lough)

**PlatformIO:**
```ini
lib_deps =
    wemos/UniversalTelegramBot @ ^1.3.0
```

### 5. Загрузка прошивки

1. Подключи ESP32 к компьютеру
2. Выбери плату: `ESP32 Dev Module`
3. Нажми **Загрузить**

## 🎮 Использование

1. **Питание**: Подключи ESP32 к USB или батарее
2. **Автозапуск**: Устройство автоматически начнет работу
3. **Калибровка**: Первые 2 секунды измеряется уровень шума
4. **Запись**: При звуке выше порога записывается 3 секунды
5. **Отправка**: Аудио автоматически отправляется в Telegram

### Индикация

- 🔴 **Моргание светодиода** — идет запись
- 📱 **Сообщение в Telegram** — подтверждение готовности

## ⚙️ Настройка параметров

```cpp
#define SAMPLE_RATE 8000      // Частота дискретизации (Гц)
#define RECORD_SECONDS 3      // Длительность записи (сек)
```

**Рекомендации:**
- `8000 Гц` — оптимально для голоса
- `16000 Гц` — лучшее качество (больше памяти)
- `2-3 секунды` — достаточно для команды

## 🛠️ Возможные проблемы и решения

### ESP32 не подключается к WiFi
✅ Проверь SSID и пароль  
✅ Убедись, что сеть работает на 2.4GHz  

### Не отправляются сообщения
✅ Проверь токен бота  
✅ Убедись, что бот не заблокирован  
✅ Проверь Chat ID  

### Запись слишком тихая
✅ Уменьши порог `threshold = noiseLevel + 150`  
✅ Говори ближе к микрофону  

## 🔒 Безопасность

- 🔐 Токен бота хранится в коде — не публикуй его!
- 📡 Соединение с Telegram защищено TLS
- 💾 Аудио хранится только в Telegram
